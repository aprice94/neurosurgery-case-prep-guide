# Figure Library

Reusable schematic diagrams (SVG, self-contained, render on GitHub and in any browser) shared across the case guides. Each guide's **Position** / **Approach** sections link to the relevant figure here rather than duplicating art.

Style: flat, labeled line-schematics for **learning case set-up** — body orientation, head fixation, supports/rolls, table tilt, danger zones, incision/craniotomy outlines.

## How to Use This Library
1. **Pre-op prep:** Open the relevant case guide; review positioning, approach, critical anatomy, and key steps.
2. **Build a patient-specific note:** Copy the guide's **Op Note Template** and fill in the bracketed `[ ]` fields with the specific patient's details (the one-liner, PMH, imaging, labs, exam, and planning sections are designed to become a complete op note).
3. **References:** Use [shunt-systems-reference](cases/cranial-csf/shunts/shunt-systems-reference.md) and [biopsy-systems-reference](cases/biopsy/biopsy-systems-reference.md) for device/platform selection.
4. **Resource lookup:** [neurosurgery-resources.json](resources/neurosurgery-resources.json) for the source textbooks/atlases/videos behind each approach.

> **Disclaimer:** These are personal study/preparation aids summarizing standard teaching. Always verify against primary sources, current guidelines, institutional protocols, and attending/operative judgment for any specific patient. See [DISCLAIMER.md](DISCLAIMER.md).
